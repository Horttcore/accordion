# Accordion

A simple accordion block

## Installation

Install via composer
- `$ composer require ralfhortt/accordion-block`

Import the block
- `import "../../../vendor/ralfhortt/accordion";`

## Features

- Native html elements (details / summary)
- Title
- InnerBlocks

## Usage

### Keywords

- Accordion
- Toggle
